# Terminología


## Minería de datos (*Data Mining*)

La minería de datos es el proceso de descubrimiento de patrones en grandes conjuntos de datos. La minería de datos se utiliza en el aprendizaje automático para descubrir patrones y relaciones en los datos, y para predecir el comportamiento futuro de los sistemas.


## KDD

Knowledge Discovery in Databases (KDD) es el proceso de descubrimiento de conocimiento útil a partir de grandes volúmenes de datos almacenados en bases de datos. El proceso de KDD consta de varias fases:
- Preprocesamiento de datos
- Selección de datos
- Transformación de datos
- Minería de datos
- Interpretación de los resultados
