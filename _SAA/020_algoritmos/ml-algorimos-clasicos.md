## Aprendizaje supervisado

### Regresión lineal

La regresión lineal es un modelo de aprendizaje supervisado que se utiliza para predecir valores numéricos. Es decir, se utiliza para predecir un valor numérico dependiente de otra variable.

La regresión lineal es un modelo lineal, es decir, un modelo que asume que la relación entre las variables de entrada $x$ y la variable de salida $y$ es lineal. Esto significa que el modelo puede representarse como una línea recta.

La regresión lineal se puede utilizar para resolver problemas de regresión, es decir, problemas en los que se predice un valor numérico. Por ejemplo, se puede utilizar para predecir el precio de una casa en función de su tamaño y ubicación.
